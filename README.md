# My DSA Code

This repository contains C++ implementations of core Data Structures and Algorithms (DSA) topics for learning and interview practice.

## Overview

The codebase is organized topic-wise so you can quickly navigate to a specific concept and run examples.

Status: These topics are prepared up to now and will be updated with more problems regularly.

## Directory Structure

- `LinkedList`: Basic linked list operations and problems
- `arrays`: Array-based algorithms and practice problems
- `graphs`: Graph traversal and related algorithms (BFS, DFS, etc.)
- `hashing`: Hashing concepts and problem solutions
- `heap`: Heap implementation and heap-based problems
- `matrices`: Matrix operations and matrix-related algorithms
- `queue`: Queue implementation and practice problems
- `recursion`: Recursive patterns and recursive problem solving
- `sorting`: Sorting algorithms (QuickSort, MergeSort, etc.)
- `stack`: Stack implementation and stack-based problems
- `strings`: String algorithms and manipulation problems
- `structures`: Additional data structure implementations
- `template`: Reusable C++ templates/snippets
- `tree`: Tree data structures (Binary Tree, BST, etc.)
- `vectors`: Vector usage and vector-based problems
